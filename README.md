eredax
=============

SL commuter traffic stop information

Dependencies
--------------------
* nodejs
* expressjs
* json-proxy



Fetch all dependencies with npm:
```
npm install
```

Configure
---------
Get API keys: https://www.trafiklab.se

Edit `example.config.js` (rename to <b>config.js</b>)

Run
---
`node server.js` or `npm start`
