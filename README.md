eredax
=============

SL commuter traffic stop information

Dependencies
--------------------
* nodejs
* expressjs
* json-proxy



Fetch all dependencies with npm:
```
npm install
```

Configure
---------
Get API keys: https://www.trafiklab.se

Edit `/app/example.conf` (rename to <b>app.conf</b>)

Run
---
`node start.js` or `npm start`
